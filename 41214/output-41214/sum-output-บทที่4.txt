
# บทที่ 4: โมฆะกรรมและโมฆียะกรรม

บทนี้กล่าวถึงความไม่สมบูรณ์ของนิติกรรมในสองลักษณะ คือ **โมฆะกรรม** (Void) และ **โมฆียะกรรม** (Voidable) พร้อมสาเหตุ ผลทางกฎหมาย และวิธีจัดการกับนิติกรรมที่ขาดความสมบูรณ์  

## 🎯 ความแตกต่างระหว่างโมฆะกรรมและโมฆียะกรรม
- **โมฆะกรรม** คือ นิติกรรมที่เสียเปล่าตั้งแต่เริ่ม ไม่ก่อให้เกิดผลทางกฎหมายใด ๆ ทั้งสิ้น แม้คู่กรณีจะปฏิบัติตาม ก็ไม่เกิดผลผูกพัน.
- **โมฆียะกรรม** คือ นิติกรรมที่สมบูรณ์และมีผลผูกพันตั้งแต่ต้น แต่อาจถูกบอกล้างภายหลังโดยผู้มีสิทธิบอกล้าง เมื่อบอกล้างแล้วนิติกรรมกลายเป็นโมฆะ หากไม่มีการบอกล้าง หรือมีการให้สัตยาบัน นิติกรรมจะสมบูรณ์ตลอดไป.
- ความแตกต่างสำคัญ:
  - โมฆะกรรม **ไม่ต้องบอกล้าง** เพราะไม่มีผลตั้งแต่ต้น; โมฆียะกรรม **ต้องบอกล้าง** จึงจะสิ้นผล.
  - โมฆะกรรม **ไม่อาจให้สัตยาบัน**; โมฆียะกรรม **ให้สัตยาบันได้**.
  - ทุกคนที่มีส่วนได้เสียยกเรื่องโมฆะกรรมขึ้นอ้างได้ตลอดเวลา; โมฆียะกรรมจะบอกล้างได้เฉพาะผู้ที่กฎหมายให้สิทธิเท่านั้น ภายในเวลาที่กำหนด.
  - โมฆะกรรมไม่มีบทบัญญัติให้คู่กรณีคืนสู่ฐานเดิม; การคืนทรัพย์ต้องอาศัยหลัก **ลาภมิควรได้**; ส่วนโมฆียะกรรม เมื่อตกเป็นโมฆะเพราะบอกล้าง คู่กรณีคืนสู่ฐานเดิมได้.

## 🚫 เหตุให้เป็นโมฆะกรรม (โมฆะเพราะขัดหลักกฎหมาย)
1. **ขัดต่อบทบัญญัติหรือแบบของกฎหมาย**
   - เช่น ขายที่ดินโดยไม่ทำเป็นหนังสือและจดทะเบียนตามมาตรา 456 → โมฆะ.
   - สัญญาก่อนสมรสไม่ได้ปฏิบัติตามมาตรา 1466; พินัยกรรมฝ่าฝืนมาตรา 1705-1706 → โมฆะ.

2. **วัตถุประสงค์ต้องห้ามชัดแจ้ง, พันวิสัย, ขัดความสงบเรียบร้อยหรือศีลธรรม**
   - ซื้อขายยาบ้า, จ้างฆ่า → โมฆะเพราะขัดกฎหมายและศีลธรรม.
   - วัตถุประสงค์เป็นไปไม่ได้ (พันวิสัย) เช่น โอนทรัพย์ที่สูญหายไปแล้ว; หรือกฎหมายห้ามซื้อขายทรัพย์ชนิดนั้น → โมฆะ.
   - สัญญาฝากเข้าทำงานโดยใช้เส้น (จ่ายเงินให้ข้าราชการ) ขัดต่อความสงบเรียบร้อย → โมฆะ.

3. **ไม่ถูกต้องตามแบบ**
   - กฎหมายกำหนดให้ทำเป็นหนังสือและ/หรือจดทะเบียน; หากไม่ทำ เช่น ซื้อขายอสังหาริมทรัพย์, จดทะเบียนเครื่องหมายการค้า → โมฆะ.

4. **แสดงเจตนาลวง/อำพราง**
   - เจตนาลวงร่วมกันเพื่อไม่ผูกพันจริง (แกล้งขายทรัพย์หลบหนี้) → โมฆะ แต่หากบุคคลภายนอกซื้อโดยสุจริตย่อมไม่เสียสิทธิ.
   - นิติกรรมอำพราง เช่น ทำสัญญาซื้อขายเพื่อซ่อนการให้; นิติกรรมที่ถูกอำพรางมีผลหากทำถูกต้องตามแบบและวัตถุประสงค์เดียวกัน.

5. **สัญญามีเงื่อนไขต้องห้าม**
   - กำหนดเงื่อนไขบังคับหลังที่สำเร็จแล้วแต่เวลาทำ หรือเงื่อนไขบังคับก่อนที่รู้แน่ว่าไม่อาจสำเร็จ; เงื่อนไขขัดกฎหมายหรือศีลธรรม; เงื่อนไขให้สิทธิแก่ลูกหนี้โดยฝ่ายเดียว → ทำให้สัญญาเป็นโมฆะ (มาตรา 187–190).

## 📌 ผลของโมฆะกรรม
- ไม่มีผลผูกพันแต่แรก; ทุกคนที่มีส่วนได้เสียอ้างได้ตลอดโดยไม่มีอายุความ.
- การคืนทรัพย์สินให้ทำในฐานะ **ลาภมิควรได้** ตามมาตรา 406–419 เพราะไม่มีสิทธิเรียกร้องตามสัญญา.
- ไม่สามารถให้สัตยาบัน; หากจะทำให้นิติสัมพันธ์เกิดผล ต้องทำสัญญาใหม่ที่ถูกต้องตามกฎหมาย.

## ♻️ โมฆียะกรรม: เหตุและผล
โมฆียะกรรมเกิดจากกฎหมายต้องการคุ้มครองผู้ที่บกพร่องความสามารถหรือแสดงเจตนาวิปริต ให้มีโอกาสยืนยันหรือยกเลิกนิติกรรมได้ โดยมีลักษณะและผลดังนี้:

### 🎗️ เหตุทำให้เป็นโมฆียะกรรม
1. **ผู้ทำขาดความสามารถบางส่วน**
   - *ผู้เยาว์* (อายุต่ำกว่า 20 ปี), *คนเสมือนไร้ความสามารถ* และ *คนไร้ความสามารถ* ที่ทำสัญญาโดยไม่ได้รับความยินยอมจากผู้แทนโดยชอบธรรม → โมฆียะ (ผู้แทนหรือศาลอาจบอกล้างได้).  
   - สัญญาฝึกงาน, สัญญาให้, การฝากทรัพย์อาจมีสถานะต่างกันตามกฎหมายพิเศษ.

2. **แสดงเจตนาโดยสำคัญผิด**
   - *ในสาระสำคัญของนิติกรรม* เช่น เข้าใจผิดว่าสัญญาที่ลงนามเป็นสัญญากู้ แต่จริงเป็นสัญญาซื้อขายที่ดิน → โมฆะ (มาตรา 156).  
   - *ในตัวบุคคล* เมื่อบุคคลเป็นสาระสำคัญ เช่น สัญญายืม, เช่าทรัพย์, จ้างแรงงาน; ถ้าเข้าใจผิดว่าอีกฝ่ายเป็นคนอื่น → โมฆะ.  
   - *ในคุณสมบัติของบุคคลหรือทรัพย์* (ของแท้ vs ของเทียม); นิติกรรมเป็นโมฆียะ ผู้ผิดอาจบอกล้างได้ภายในกำหนดเวลา.

3. **ถูกกลฉ้อฉล (หลอกลวง)**
   - หลอกจนทำให้คู่กรณีตกลง ทั้งที่หากรู้ความจริงคงไม่ทำสัญญา → โมฆียะ; ผู้ถูกหลอกมีสิทธิบอกล้าง.  
   - ถ้ากลฉ้อฉลมาจากบุคคลภายนอก นิติกรรมเป็นโมฆียะต่อเมื่อคู่กรณีอีกฝ่าย *รู้หรือควรรู้* ว่ามีการหลอกลวง.  
   - กลฉ้อฉลที่ไม่ถึงขนาดหรือผู้เสียหายควรตรวจสอบเอง เช่น โฆษณาเกินจริงที่ผู้ซื้อควรระมัดระวัง จะไม่ทำให้นิติกรรมเป็นโมฆียะ.

4. **ถูกข่มขู่**
   - ข่มขู่ด้วยภัยใกล้จะถึงและร้ายแรงจนจูงใจให้ทำสัญญา → โมฆียะ; ผู้ถูกข่มขู่มีสิทธิบอกล้าง (มาตรา 164).  
   - การขู่ว่าจะใช้สิทธิตามกฎหมาย เช่น ฟ้องร้องตามสัญญา หรือการนับถือยำเกรงผู้มีอำนาจในครอบครัว ไม่ถือเป็นการข่มขู่ จึงไม่ทำให้สัญญาเป็นโมฆียะ.

### 🔄 การบอกล้างและการให้สัตยาบัน
- **บอกล้าง (Annulment)**
  - ผู้มีสิทธิบอกล้าง ได้แก่ คู่กรณีที่กฎหมายคุ้มครอง เช่น ผู้แทนโดยชอบธรรมของผู้เยาว์, ผู้ถูกหลอก, ถูกข่มขู่, หรือผู้รู้ภายหลังว่ามีการสำคัญผิด.  
  - ต้องแสดงเจตนาบอกล้างต่ออีกฝ่าย **ภายใน 1 ปี** นับแต่วันที่หมดเหตุบกพร่อง และไม่เกิน **10 ปี** นับแต่ทำนิติกรรม (มาตรา 181).  
  - เมื่อบอกล้าง นิติกรรมเป็นโมฆะย้อนหลัง; คู่กรณีต้องคืนสู่ฐานะเดิม และอาจเรียกร้องค่าเสียหายตามกฎหมาย.

- **การให้สัตยาบัน (Ratification)**
  - ผู้มีสิทธิบอกล้างหรือผู้แทนโดยชอบธรรมอาจรับรองนิติกรรมด้วยวาจา เป็นหนังสือ หรือกระทำการใดแสดงว่าต้องการให้นิติกรรมมีผลต่อไป (มาตรา 178–179).  
  - เมื่อให้สัตยาบันแล้ว นิติกรรมนั้นสมบูรณ์ตลอดไป ไม่อาจบอกล้างได้อีก.

### 🎁 ผลของโมฆียะกรรม
- นิติกรรมมีผลบังคับจนกว่าจะถูกบอกล้าง; ผู้บอกล้างต้องทำภายในกำหนดเวลา.  
- เมื่อถูกบอกล้าง คู่กรณีต้องคืนสภาพเหมือนก่อนทำสัญญา; อาจเรียกร้องค่าเสียหายแทนการคืนทรัพย์ไม่ได้ เช่น กรณีทรัพย์สินพินาศ.  
- การให้สัตยาบันทำให้นิติกรรมมีผลสมบูรณ์ถาวร.

## 📝 เคล็ดลับจำก่อนสอบ
- แยกให้ออก **โมฆะ (Void)** และ **โมฆียะ (Voidable)** โดยดูว่า “ไม่มีผลเลย” vs “มีผลจนกว่าจะบอกล้าง”.
- จำสาเหตุของการเป็นโมฆะกรรม: ขัดกฎหมายหรือศีลธรรม, พันวิสัย, แบบไม่ถูกต้อง, เจตนาลวง/อำพราง, เงื่อนไขต้องห้าม.
- จำสาเหตุของโมฆียะกรรม: ความสามารถบกพร่อง (ผู้เยาว์), สำคัญผิด (ลักษณะนิติกรรม/บุคคล/คุณสมบัติ), ถูกหลอก, ถูกข่มขู่.
- รู้ระยะเวลาบอกล้าง (1 ปี – ไม่เกิน 10 ปี) และเงื่อนไขการให้สัตยาบัน.
- ใช้ **ตัวอย่างคำพิพากษาศาลฎีกา** เพื่อเห็นภาพจริงในการสอบ เช่น ขายที่ดินโดยไม่จดทะเบียน → โมฆะ; ซื้อรถโดยหลงเชื่อลักษณะสินค้า → โมฆียะ; การขู่จะใช้สิทธิในศาลไม่ถือเป็นข่มขู่ เป็นต้น.

หวังว่าบทสรุปนี้จะช่วยให้เข้าใจความแตกต่างและรายละเอียดของโมฆะกรรมและโมฆียะกรรมได้ชัดเจน พร้อมเตรียมตัวทำข้อสอบได้อย่างมั่นใจ 😊
