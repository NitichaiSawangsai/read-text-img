🔴 เรื่องที่ 1 — ข่มขืนกระทำชำเรา (มาตรา 276) ★★★ ออกสอบบ่อยมาก

▶ หลักกฎหมาย
มาตรา 276 วรรคหนึ่ง: "ผู้ใดข่มขืนกระทำชำเราผู้อื่น โดยขู่เข็ญ / ใช้กำลังประทุษร้าย / ผู้อื่นอยู่ในภาวะที่ไม่สามารถขัดขืนได้ / ทำให้เข้าใจผิดว่าตนเป็นบุคคลอื่น"
→ โทษ: จำคุก 4–20 ปี ปรับ 80,000–400,000 บาท

▶ ใครเป็นผู้กระทำได้บ้าง?
• ชายข่มขืนหญิง ✓
• หญิงข่มขืนชาย ✓
• ชายข่มขืนชาย ✓
• หญิงข่มขืนหญิง ✓
• สามีข่มขืนภริยา หรือภริยาข่มขืนสามี ✓ (แก้ไขจากกฎหมายเดิมที่ยกเว้นกรณีนี้)

▶ คำว่า "กระทำชำเรา" (มาตรา 1(18))
= การใช้อวัยวะเพศของผู้กระทำล่วงล้าอวัยวะเพศ ทวารหนัก หรือช่องปากของผู้อื่น "เพื่อสนองความใคร่ของผู้กระทำ"
• ต้องสอดใส่เข้าไปได้ ถ้าแค่สัมผัสหรือถูไถ → ไม่เป็นการกระทำชำเราสำเร็จ → อาจเป็นพยายามหรืออนาจาร

▶ 4 วิธีที่ทำให้เป็นความผิด
(1) โดยขู่เข็ญด้วยประการใดๆ — ทำให้กลัวว่าจะเกิดอันตราย ไม่ต้องมีอาวุธก็ได้
(2) โดยใช้กำลังประทุษร้าย — ทำร้ายกายหรือจิตใจ รวมถึงการใช้ยาทำให้มึนเมา/สะกดจิต
(3) อยู่ในภาวะที่ไม่สามารถขัดขืนได้ — เช่น เป็นอัมพาต นอนหลับ เมาสุราหมดสติ
(4) ทำให้เข้าใจผิดว่าตนเป็นบุคคลอื่น — เช่น แอบสวมรอยเป็นสามี/คนรัก

▶ ข้อสำคัญ: "ยินยอม" ต้องเป็นการยินยอมโดยสมัครใจจริงๆ
• ยินยอมเพราะกลัวถูกฆ่า หรือถูกทำร้าย → ถือว่าไม่ยินยอม → ยังผิด
• ยินยอมเพราะถูกหลอกว่าจะแต่งงาน → ถือว่ายินยอมโดยสมัครใจ → ไม่ผิดมาตรานี้

▶ เหตุรับโทษหนักขึ้น
• ใช้อาวุธปืน/วัตถุระเบิด หรือ ทำให้เชื่อว่ามีอาวุธ → โทษหนักขึ้น
• โทรมหญิง/โทรมชาย (ร่วมกัน 2 คนขึ้นไปผลัดกันกระทำ) → จำคุก 15–20 ปี หรือตลอดชีวิต
• เป็นเหตุให้ได้รับอันตรายสาหัส → จำคุก 15–20 ปี หรือตลอดชีวิต (ม.277 ทวิ)
• เป็นเหตุให้ถึงแก่ความตาย → ประหารชีวิต หรือจำคุกตลอดชีวิต (ม.277 ทวิ)

▶ รับโทษหนักอีก 1/3 ถ้ากระทำต่อ (มาตรา 285)
บุพการี / ผู้สืบสันดาน / พี่น้องร่วมสายเลือด / ศิษย์ที่อยู่ในความดูแล / ผู้อยู่ในความควบคุมตามหน้าที่ราชการ / ผู้อยู่ในความปกครอง/พิทักษ์/อนุบาล

▶ รับโทษหนักอีก 1/3 ถ้ากระทำต่อผู้ที่ปกป้องตัวเองไม่ได้ (มาตรา 285/2)
เช่น ผู้ทุพพลภาพ / จิตบกพร่อง / โรคจิต / คนป่วยเจ็บ / คนชรา / สตรีมีครรภ์

▶ รับโทษหนักอีก 1/3 ถ้าบันทึกภาพ/เสียง เพื่อแสวงหาประโยชน์โดยมิชอบ (มาตรา 280/1)
→ ถ้าเผยแพร่/ส่งต่อ → โทษหนักขึ้นอีกกึ่งหนึ่ง

▶ ความผิดอันยอมความได้ (มาตรา 281)
กรณียอมความได้: ข่มขืนระหว่างคู่สมรส ที่ไม่เกิดต่อหน้าธารกำนัล / ไม่เป็นเหตุให้อันตรายสาหัสหรือตาย / ไม่ได้กระทำต่อบุคคลตามมาตรา 285, 285/2

▶ กรณีคู่สมรสกระทำต่อกัน (มาตรา 276 วรรคสี่)
• คู่สมรสยังอยู่ด้วยกัน → ศาลลงโทษน้อยกว่าที่กฎหมายกำหนดได้ หรือกำหนดเงื่อนไขคุมประพฤติแทน
• คู่สมรสประสงค์หย่า → ศาลแจ้งพนักงานอัยการให้ดำเนินการฟ้องหย่าให้

🔴 เรื่องที่ 2 — กระทำชำเราเด็ก (มาตรา 277) ★★★ ออกสอบบ่อยมาก

▶ หลักกฎหมาย
มาตรา 277: "ผู้ใดกระทำชำเราเด็กอายุยังไม่เกิน 15 ปี ซึ่งมิใช่ภริยาหรือสามีของตน โดยเด็กนั้นจะยินยอมหรือไม่ก็ตาม"
→ โทษ: จำคุก 5–20 ปี ปรับ 100,000–400,000 บาท

▶ ข้อสำคัญ: ไม่ต้อง "ข่มขืน" — แม้เด็กยินยอม ก็ผิด!
เหตุผล: เด็กอายุ ≤ 15 ปี ยังไม่สามารถรู้ผิดชอบเรื่องทางเพศ

▶ ยกเว้นความผิดถ้า: เด็กเป็น "ภริยา/สามี" โดยชอบตามกฎหมายอิสลาม หรือศาลอนุญาตให้สมรส (ตาม ปพพ. ม.1448)

▶ โทษหนักขึ้น
• เด็กอายุ ≤ 13 ปี → จำคุก 7–20 ปี หรือตลอดชีวิต
• ใช้/ทำให้เชื่อว่ามีอาวุธ → โทษหนักขึ้น
• โทรมเด็กหญิง/กระทำกับเด็กชายในลักษณะเดียวกัน → จำคุกตลอดชีวิต
• กระทำต่อเด็ก ≤ 13 ปี → ห้ามอ้างว่าไม่รู้อายุ (มาตรา 285/1)

▶ ได้ยกเว้นโทษหรือรับโทษน้อย (มาตรา 277 วรรคห้า–หก)
เงื่อนไข: ผู้กระทำอายุ ≤ 18 ปี / กระทำต่อเด็ก 13–15 ปี / เด็กยินยอม
→ ศาลอาจให้มีการคุ้มครองสวัสดิภาพแทนการลงโทษ / ถ้าสำเร็จ = ไม่ต้องรับโทษ

🔴 เรื่องที่ 3 — กระทำอนาจาร (มาตรา 278–279) ★★

▶ "กระทำอนาจาร" = การกระทำแก่เนื้อตัวของบุคคล ที่ไม่สมควรในทางเพศตามประเพณีนิยม แต่ยังไม่ถึงขั้นกระทำชำเรา
• เช่น: กอด จูบ จับนม จับของลับ เลิกกระโปรง ล้วงมือเข้ากระโปรง
• ไม่ใช่: พูดหยาบคาย เขียนภาพลามก (ไม่ถือเป็นอนาจาร)

▶ มาตรา 278 — อนาจารต่อบุคคลอายุ > 15 ปี (ไม่ยินยอม)
→ จำคุก ≤ 10 ปี หรือปรับ ≤ 200,000 บาท
→ ถ้าใช้วัตถุ/อวัยวะอื่น (ไม่ใช่อวัยวะเพศ) ล่วงล้าอวัยวะเพศ/ทวารหนัก → จำคุก 4–20 ปี

▶ มาตรา 279 — อนาจารต่อเด็กอายุ ≤ 15 ปี (แม้ยินยอม)
→ จำคุก ≤ 10 ปี หรือปรับ ≤ 200,000 บาท
→ เด็กอายุ ≤ 13 ปี → โทษหนักขึ้น

▶ ความผิดอันยอมความได้ (มาตรา 281)
• กระทำอนาจารตาม ม.278 ว.1 ที่ไม่เกิดต่อหน้าธารกำนัล / ไม่เป็นเหตุให้อันตรายสาหัส / ไม่ใช่บุคคลตาม ม.285, 285/2

🔴 เรื่องที่ 4 — เป็นธุระจัดหา ล่อไป พาไป เพื่อการอนาจาร ★★

▶ มาตรา 282 — จัดหาชาย/หญิง/เด็กเพื่อสนองความใคร่ผู้อื่น (แม้ยินยอม)
• ต้องมีเจตนาพิเศษ "เพื่อสนองความใคร่ผู้อื่น" และ "เพื่อการอนาจาร"
• โทษหนักขึ้นถ้ากระทำต่อ: บุคคลอายุ 15–18 ปี หรือเด็ก ≤ 15 ปี

▶ มาตรา 283 — จัดหาชาย/หญิง/เด็กโดยไม่ยินยอม
• วิธี: ใช้อุบายหลอกลวง / ขู่เข็ญ / ใช้กำลัง / ใช้อำนาจครอบงำผิดครองธรรม / ข่มขืนใจ
• โทษ: จำคุก 5–20 ปี ปรับ 100,000–400,000 บาท
• ถ้าเด็ก ≤ 15 ปี → โทษหนักขึ้น / ถ้าเด็ก ≤ 13 ปี → ห้ามอ้างไม่รู้อายุ

▶ มาตรา 283 ทวิ — พาบุคคลอายุ 15–18 ปี ไปเพื่ออนาจาร (แม้ยินยอม)
→ จำคุก ≤ 5 ปี หรือปรับ ≤ 100,000 บาท

▶ มาตรา 284 — พาผู้อื่นไปเพื่ออนาจาร (ไม่ยินยอม) โดยใช้อุบาย/ขู่เข็ญ/กำลัง
→ จำคุก 1–10 ปี ปรับ 20,000–200,000 บาท (ยอมความได้)

▶ ผู้รับตัว และผู้สนับสนุน (มาตรา 282 วรรคสี่, มาตรา 283 วรรคสี่)
→ รับโทษเท่ากับตัวการ (ไม่ใช่เพียง 2/3 ตาม ม.86)

🔴 เรื่องที่ 5 — ช่วยเหลือการค้าประเวณี (มาตรา 286) ★

มาตรา 286 กำหนดความผิด 7 ลักษณะ เช่น
(1) ช่วยเหลือ/ให้ความสะดวก/คุ้มครองการค้าประเวณีผู้อื่น
(2) รับประโยชน์จากการค้าประเวณีของผู้อื่น
(6) อยู่ร่วมกับผู้ค้าประเวณีเป็นอาจิณ และไม่สามารถแสดงที่มาของรายได้ในการดำรงชีพ
→ โทษ: จำคุก ≤ 20 ปี หรือตลอดชีวิต ปรับ ≤ 400,000 บาท

⚠️ ข้อยกเว้น: ผู้รับประโยชน์โดยชอบตามกฎหมาย/ธรรมจรรยา ไม่ผิด
(เช่น ได้รับค่าเลี้ยงดูตาม ปพพ. หรือในฐานะเครือญาติ)

🔴 เรื่องที่ 6 — สื่อลามก/สื่อลามกอนาจารเด็ก (มาตรา 287, 287/1, 287/2) ★

▶ มาตรา 287 — ค้าวัตถุลามก
→ จำคุก ≤ 3 ปี หรือปรับ ≤ 60,000 บาท

▶ มาตรา 287/1 — ครอบครองสื่อลามกอนาจารเด็ก เพื่อแสวงหาประโยชน์ทางเพศ
→ จำคุก ≤ 5 ปี หรือปรับ ≤ 100,000 บาท
→ ถ้าส่งต่อ → จำคุก ≤ 7 ปี

▶ มาตรา 287/2 — ผลิต/เผยแพร่สื่อลามกอนาจารเด็ก
→ จำคุก 3–10 ปี ปรับ 60,000–200,000 บาท

"สื่อลามกอนาจารเด็ก" (มาตรา 1(17)) = วัตถุที่แสดงการกระทำทางเพศของเด็กหรือกับเด็กอายุ ≤ 18 ปี

🔵 แนวคิดสำคัญ

▶ "แท้งลูก" = การทำลายทารกในครรภ์มารดา ตั้งแต่เริ่มปฏิสนธิจนถึงก่อนที่ทารกจะมีสภาพบุคคล
• ทารกมีสภาพบุคคลเมื่อ "คลอดแล้วอยู่รอดเป็นทารก" (แสดงอาการมีชีวิต เช่น หัวใจเต้น เคลื่อนไหว)
• ถ้าทารกยังอยู่ในครรภ์/กำลังคลอดแต่ยังไม่มีสภาพบุคคล → ผิดฐานทำให้แท้งลูก
• ถ้าทารกมีสภาพบุคคลแล้ว → ผิดฐานฆ่าผู้อื่น

▶ เส้นแบ่ง: การทำให้แท้งลูก ≠ การทำให้คลอดก่อนกำหนดที่ทารกยังมีชีวิตรอด

🔵 เรื่องที่ 1 — หญิงทำให้ตนเองแท้งลูก (มาตรา 301) ★★

▶ หลักกฎหมาย
มาตรา 301: หญิงทำให้ตนเองแท้งลูก หรือยอมให้ผู้อื่นทำให้ตนแท้งลูก "ขณะมีอายุครรภ์เกิน 12 สัปดาห์"
→ โทษ: จำคุก ≤ 6 เดือน หรือปรับ ≤ 10,000 บาท

⚠️ สำคัญ: ถ้าอายุครรภ์ ≤ 12 สัปดาห์ → หญิงไม่ผิด แต่ผู้อื่นที่ช่วยทำให้แท้ง ยังผิดตามมาตรา 302

▶ หลักการแบ่งความรับผิด
• ถ้าหญิงร่วมกับผู้อื่นทำ → หญิงผิดตาม ม.301, ผู้อื่นผิดตาม ม.302
• ถ้าผู้อื่นแนะนำ/สนับสนุนให้หญิงทำเอง → ผู้อื่นเป็นผู้สนับสนุน (ม.301 + ม.86) ไม่ใช่ผู้ใช้

---

🔵 เรื่องที่ 2 — ผู้อื่นทำให้หญิงแท้งลูก (มาตรา 302–303) ★★★ ออกสอบบ่อยมาก

▶ มาตรา 302 — ทำให้หญิงแท้งลูกโดยหญิงยินยอม
→ จำคุก ≤ 5 ปี หรือปรับ ≤ 100,000 บาท
→ ถ้าเป็นเหตุให้หญิงรับอันตรายสาหัส → จำคุก ≤ 7 ปี
→ ถ้าเป็นเหตุให้หญิงถึงแก่ความตาย → จำคุก ≤ 10 ปี ปรับ ≤ 200,000 บาท

▶ มาตรา 303 — ทำให้หญิงแท้งลูกโดยหญิงไม่ยินยอม
→ จำคุก ≤ 7 ปี หรือปรับ ≤ 140,000 บาท
→ ถ้าเป็นเหตุให้หญิงรับอันตรายสาหัส → จำคุก 1–10 ปี
→ ถ้าเป็นเหตุให้หญิงถึงแก่ความตาย → จำคุก 5–20 ปี

▶ หลักเกี่ยวกับการกระทำ
• การกระทำ ไม่จำกัดวิธี: กินยา ฉีดยา ใช้กำลังกดท้อง ใช้เครื่องมือ ผ่าตัด
• เกิดจากการกระทำ ไม่ใช่การละเว้น
• ความผิดสำเร็จเมื่อทารกในครรภ์ดาย ถ้าทารกไม่ตาย = ผิดพยายาม แต่ได้รับยกเว้นโทษตาม ม.304 ในกรณีที่หญิงยินยอม

---

🔵 เรื่องที่ 3 — การทำแท้งที่ไม่ต้องรับโทษ และโดยชอบด้วยกฎหมาย (มาตรา 304–305) ★

▶ มาตรา 304 — พยายามทำแท้งที่ได้รับยกเว้นโทษ
→ กรณีผู้กระทำเป็นหญิงมีครรภ์หรือหญิงยินยอม: พยายามทำแท้งไม่ต้องรับโทษ

▶ มาตรา 305 — ทำแท้งโดยชอบด้วยกฎหมาย
(1) กระทำโดยผู้ประกอบวิชาชีพเวชกรรม
(2) หญิงยินยอม
(3) เป็นไปตามหลักเกณฑ์ที่กฎหมายกำหนด เช่น จำเป็นทางการแพทย์ หรืออายุครรภ์ ≤ 12 สัปดาห์ในกรณีที่กำหนด

🟢 เรื่องที่ 1 — ทอดทิ้งเด็ก คนป่วยเจ็บ หรือคนชรา (มาตรา 306–307) ★★

▶ มาตรา 306 — ทอดทิ้งเด็กอายุยังไม่เกิน 9 ปี
"ผู้ใดทอดทิ้งเด็กอายุยังไม่เกิน 9 ปี ไว้ ณ สถานที่ใด..."
→ โทษ: จำคุก ≤ 3 ปี หรือปรับ ≤ 60,000 บาท

→ ถ้าการทอดทิ้งเป็นเหตุให้เด็กได้รับอันตรายแก่กายหรือจิตใจ → จำคุก ≤ 5 ปี ปรับ ≤ 100,000 บาท
→ ถ้าถึงแก่ความตาย → จำคุก ≤ 7 ปี ปรับ ≤ 140,000 บาท

▶ มาตรา 307 — ทอดทิ้งคนที่ต้องดูแล
"ผู้ใดมีหน้าที่ตามกฎหมายหรือสัญญาต้องดูแลผู้ซึ่งพึ่งตนเองมิได้ เพราะอายุ ป่วยเจ็บ / กายพิการ / หรือจิตพิการ ทิ้งให้บุคคลนั้นไป..."
→ โทษ: จำคุก ≤ 3 ปี หรือปรับ ≤ 60,000 บาท

▶ หลักสำคัญ: ต้องมี "หน้าที่ดูแล" ตามกฎหมายหรือสัญญา
• พ่อแม่ทอดทิ้งลูก ✓
• แพทย์/พยาบาลทอดทิ้งคนไข้ก็อาจผิด (ถ้ามีสัญญาว่าจ้าง)
• ไม่มีหน้าที่ดูแล → ไม่ผิดมาตรานี้

🎯 ประเด็นที่ออกสอบบ่อย (สรุปย่อ)
════════════════════════════════════════════════════

★ ประเด็นที่ 1: "ยินยอม" แต่ไม่สมัครใจ เช่น กลัวถูกฆ่า หรือเมาหมดสติ → ถือว่าไม่ยินยอม → ยังผิดฐานข่มขืน

★ ประเด็นที่ 2: สามีข่มขืนภริยา → ผิดได้ (กฎหมายปัจจุบัน) แต่ถ้ายังอยู่ด้วยกัน → ศาลอาจลงโทษน้อยลง

★ ประเด็นที่ 3: กระทำชำเราเด็ก ≤ 15 ปี ไม่ต้องขู่เข็ญ แม้เด็กยินยอม ก็ผิด แต่ถ้าเด็กมีอายุ 13–15 ปี ผู้กระทำอายุ ≤ 18 ปี และเด็กยินยอม → ศาลอาจให้คุ้มครองสวัสดิภาพแทนโทษ

★ ประเด็นที่ 4: "โทรมหญิง" ต้องมีชาย 2 คนขึ้นไปผลัดกระทำชำเราหญิงคนเดียว ต่อเนื่องกัน แม้คนที่สองยังไม่สำเร็จ แต่ลงมือถึงขั้นพยายาม → ถือว่าโทรมหญิงแล้ว

★ ประเด็นที่ 5: ทำให้แท้งลูก ถ้าอายุครรภ์ ≤ 12 สัปดาห์ → หญิงไม่ผิด แต่ผู้ทำให้แท้งยังผิดตาม ม.302

★ ประเด็นที่ 6: ความผิดอันยอมความได้ → ต้องร้องทุกข์ภายใน 3 เดือน นับแต่รู้เรื่องและรู้ตัวผู้กระทำ

★ ประเด็นที่ 7: บันทึกภาพ/เสียงขณะกระทำชำเรา/อนาจาร เพื่อแสวงหาประโยชน์โดยมิชอบ → โทษ +1/3; เผยแพร่/ส่งต่อ → โทษ +1/2
