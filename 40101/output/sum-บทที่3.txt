📜 ที่มาของกฎหมาย

  1. จารีตประเพณี / ขนบธรรมเนียม
     - แบบแผนพฤติกรรมที่สืบทอดต่อกันมา ยอมรับร่วมกันในสังคม
     - องค์ประกอบ 5 ประการ: ความต่อเนื่อง / ยอมรับร่วมกัน / ความเป็นบรรทัดฐาน / ไม่มีผู้ตราอย่างเป็นทางการ / มีอำนาจควบคุม
     - กฎหมายไทย: มาตรา 4 วรรค 2 ป.พ.พ. ให้ใช้จารีตประเพณีเมื่อไม่มีบทกฎหมาย

  2. ศาสนา
     - หลักคำสอนทางศาสนาเป็นรากฐานกฎหมายในหลายสังคม
     - หลักศีล 5 (พุทธ) เป็นต้นกำเนิดกฎหมายอาญา (ห้ามฆ่า ลัก โกหก)

  3. รัฐธรรมนูญ — กฎหมายสูงสุด กำหนดโครงสร้างรัฐและสิทธิประชาชน

  4. ประมวลกฎหมาย — รวบรวมกฎหมายเป็นระบบลายลักษณ์อักษร

  5. กฎหมายที่ตราโดยฝ่ายนิติบัญญัติ — พระราชบัญญัติ พระราชกำหนด

  6. กฎหมายที่ตราโดยฝ่ายบริหาร — พระราชกฤษฎีกา กฎกระทรวง

  7. คำพิพากษาของศาล — วางบรรทัดฐาน (สำคัญในระบบ Common Law)

  8. ข้อเขียนของนักวิชาการทางกฎหมาย

  9. สนธิสัญญาระหว่างประเทศ

  10. ประชามติของประชาชน

  11. หลักความยุติธรรม (Equity)

🏛️ ลำดับศักดิ์กฎหมายไทย (สูง → ต่ำ)

  ชั้นที่ 1 — รัฐธรรมนูญ (กฎหมายสูงสุด)
  → กฎหมายทุกฉบับต้องไม่ขัดรัฐธรรมนูญ

  ชั้นที่ 2 — กฎหมายแม่บท
  → พระราชบัญญัติประกอบรัฐธรรมนูญ
  → พระราชบัญญัติ (พ.ร.บ.)
  → ประมวลกฎหมาย
  → พระราชกำหนด (พ.ร.ก.) — ใช้ในกรณีฉุกเฉิน

  ชั้นที่ 3 — กฎหมายลำดับรอง
  → พระราชกฤษฎีกา (พ.ร.ฎ.)
  → กฎกระทรวง
  → ประกาศกระทรวง / ระเบียบ / คำสั่ง

  ชั้นที่ 4 — ข้อบัญญัติท้องถิ่น
  → ใช้บังคับเฉพาะพื้นที่ขององค์กรปกครองส่วนท้องถิ่น

  ⚠️ หลักสำคัญ: กฎหมายระดับต่ำกว่าต้องไม่ขัดหรือแย้งต่อกฎหมายระดับสูงกว่า

★ ส่วนที่ 3: การแบ่งประเภทกฎหมาย (ออกสอบ)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📂 แบ่งตามเนื้อหา

  กฎหมายสารบัญญัติ (Substantive Law)
  → กำหนดสิทธิ หน้าที่ และความรับผิดของบุคคล
  → เช่น กฎหมายแพ่ง กฎหมายอาญา

  กฎหมายวิธีสบัญญัติ (Procedural Law)
  → กำหนดกระบวนการบังคับใช้กฎหมาย
  → เช่น ประมวลกฎหมายวิธีพิจารณาความอาญา/แพ่ง

📂 แบ่งตามลักษณะความสัมพันธ์

  กฎหมายมหาชน (Public Law)
  → ความสัมพันธ์ระหว่างรัฐกับเอกชน
  → เช่น กฎหมายรัฐธรรมนูญ กฎหมายปกครอง กฎหมายอาญา

  กฎหมายเอกชน (Private Law)
  → ความสัมพันธ์ระหว่างเอกชนด้วยกัน
  → เช่น กฎหมายแพ่งและพาณิชย์

  กฎหมายระหว่างประเทศ (International Law)
  → ความสัมพันธ์ระหว่างรัฐกับรัฐ หรือกับองค์กรระหว่างประเทศ

  - จารีตประเพณีมีองค์ประกอบ: ต่อเนื่อง / ยอมรับร่วม / เป็นบรรทัดฐาน / ไม่มีผู้ตรา / มีอำนาจควบคุม
  - ลำดับศักดิ์: รัฐธรรมนูญ > พ.ร.บ./ประมวล/พ.ร.ก. > พ.ร.ฎ. > กฎกระทรวง > ข้อบัญญัติท้องถิ่น
  - กฎหมายมหาชน = รัฐกับเอกชน; กฎหมายเอกชน = เอกชนกับเอกชน
  - สารบัญญัติ = กำหนดสิทธิ/หน้าที่; วิธีสบัญญัติ = กำหนดกระบวนการ
