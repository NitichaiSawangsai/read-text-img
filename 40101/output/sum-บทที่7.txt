📚 สรุปบทที่ 7: ระบบกระบวนการยุติธรรม
วิชา 40101 ความรู้เบื้องต้นเกี่ยวกับกฎหมายทั่วไป
=====================================================

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
★ ส่วนที่ 1: ความหมายและแนวคิดกระบวนการยุติธรรม
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚖️ กระบวนการยุติธรรม คือ
  → ขั้นตอนและวิธีดำเนินการในการบังคับให้เป็นไปตามกฎหมาย เพื่ออำนวยความยุติธรรม คุ้มครองสิทธิและเสรีภาพของบุคคล
  → ต้องสร้างสมดุลระหว่าง:
    - การรักษาความสงบเรียบร้อยของรัฐ
    - การคุ้มครองสิทธิเสรีภาพของบุคคล

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
★ ส่วนที่ 2: ประวัติกระบวนการยุติธรรมไทย (ออกสอบ)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📜 วิวัฒนาการ 4 ยุค

  สมัยสุโขทัย
  → ยังไม่มีรูปแบบชัดเจน
  → มีตระลาการทำหน้าที่พิจารณาคดี
  → มีระบบกลองร้องฎีกาให้พระมหากษัตริย์สอบสวนและตัดสิน (ต้นบัญญัติสิทธิร้องทุกข์)

  สมัยอยุธยา
  → มีระบบและความซับซ้อนมากขึ้น
  → เอกชนผู้เสียหายเป็นโจทก์ฟ้องเอง (ยังไม่มีอัยการ)
  → ใช้กฎจารีตนครบาล (ทรมานเพื่อให้สารภาพ)

  สมัยรัตนโกสินทร์ตอนต้น
  → รัชกาลที่ 1 รวบรวมกฎหมายตราสามตวง
  → ศาลกระจายอยู่ตามกรมต่างๆ ยังไม่เป็นอิสระ

  รัชกาลที่ 5 — ปฏิรูปครั้งใหญ่ (พ.ศ. 2434)
  → ตั้งศาลยุติธรรมเป็นองค์กรเฉพาะ
  → ตั้งกรมอัยการ ทำหน้าที่สอบสวนและฟ้องร้อง
  → ยกเลิกระบบไต่สวน มาใช้ระบบกล่าวหา
  → ยกเลิกวิธีจารีตนครบาล
  → ได้รับสิทธิสภาพนอกอาณาเขตกลับคืน (พ.ศ. 2481–2482)

  สมัยปัจจุบัน
  → ใช้ระบบ Civil Law
  → ระบบศาลคู่: ศาลยุติธรรม + ศาลปกครอง + ศาลรัฐธรรมนูญ

★ ส่วนที่ 3: ระบบการพิจารณาคดี 2 ระบบ (ออกสอบมาก)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📊 เปรียบเทียบระบบกล่าวหา vs ระบบไต่สวน

  ระบบกล่าวหา (Accusatorial System)
  - ใช้ใน: อังกฤษ สหรัฐฯ (Common Law)
  - คู่ความมีบทบาทสำคัญในการรวบรวมพยานหลักฐาน
  - ศาลวางตัวเป็นกลางอย่างเคร่งครัด
  - บุคคล 3 ฝ่าย: โจทก์ / จำเลย / ผู้พิพากษา
  - หลักสันนิษฐาน: จำเลยบริสุทธิ์จนกว่าจะพิสูจน์ได้
  - โจทก์มีภาระพิสูจน์ก่อน ("ผู้ใดกล่าวอ้าง ผู้นั้นต้องพิสูจน์")

  ระบบไต่สวน (Inquisitorial System)
  - ใช้ใน: ฝรั่งเศส เยอรมนี (Civil Law)
  - ศาลทำหน้าที่ค้นหาความจริงในคดี
  - ศาลมีอำนาจสืบพยานและควบคุมการไต่สวนเอง
  - ไม่เน้นความเท่าเทียมของคู่ความ
  - รัฐเป็นโจทก์ฟ้องจำเลยเสมอ
  - พยานหลักฐานเปิดกว้างตามดุลพินิจของศาล

  ⚠️ ไทยใช้: ระบบผสมผสาน — ระบบกล่าวหา (คดีแพ่ง) + ไต่สวน (คดีอาญาบางส่วน)

★ ส่วนที่ 4: ประเภทกระบวนการยุติธรรมในไทย (ออกสอบ)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚖️ กระบวนการยุติธรรมทางแพ่ง

  - ระงับข้อพิพาทเรื่องสิทธิและหน้าที่ทางแพ่งระหว่างบุคคล
  - ผู้เกี่ยวข้อง: คู่ความ ทนายความ ศาลยุติธรรม กรมบังคับคดี
  - ขั้นตอน: เตรียมการก่อนฟ้อง → ยื่นฟ้อง → พิจารณาคดี → อุทธรณ์/ฎีกา → บังคับคดี

⚖️ กระบวนการยุติธรรมทางอาญา

  - นำตัวผู้กระทำผิดมาลงโทษผ่านหน่วยงานของรัฐ
  - ผู้เกี่ยวข้อง: ตำรวจ → อัยการ → ศาล → ราชทัณฑ์

⚖️ กระบวนการยุติธรรมทางปกครอง

  - พิจารณาข้อพิพาทคดีปกครอง
  - ระบบศาลปกครอง 2 ชั้น: ศาลปกครองชั้นต้น + ศาลปกครองสูงสุด

🔄 กระบวนการยุติธรรมทางเลือก (Alternative Dispute Resolution)

  - การเจรจาต่อรอง
  - การไกล่เกลี่ย
  - อนุญาโตตุลาการ
  - กระบวนการยุติธรรมเชิงสมานฉันท์ (Restorative Justice)

★ ส่วนที่ 5: สิทธิในการเข้าถึงกระบวนการยุติธรรม
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🌐 หลักสากล

  - ปฏิญญาสากลว่าด้วยสิทธิมนุษยชน: สิทธิความเท่าเทียมภายใต้กฎหมาย
  - กติการะหว่างประเทศว่าด้วยสิทธิพลเมืองและสิทธิทางการเมือง
  - เป้าหมายการพัฒนาที่ยั่งยืน (SDGs): สิทธิเข้าถึงความยุติธรรม
  - ไทยเข้าเป็นภาคีกติการะหว่างประเทศ พ.ศ. 2539

🏛️ รัฐธรรมนูญไทย 2560
  - บัญญัติสิทธิในการเข้าถึงกระบวนการยุติธรรมอย่างชัดเจน

📌 จุดสำคัญสำหรับสอบ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  - กระบวนการยุติธรรม = อำนวยความยุติธรรม + คุ้มครองสิทธิ
  - รัชกาลที่ 5 ปฏิรูปครั้งใหญ่: ตั้งศาลยุติธรรม + กรมอัยการ + ยกเลิกจารีตนครบาล
  - ระบบกล่าวหา: ศาลเป็นกลาง คู่ความต่อสู้กันเอง (Common Law)
  - ระบบไต่สวน: ศาลค้นหาความจริงเอง (Civil Law)
  - ไทยใช้ระบบผสมผสาน
  - กระบวนการยุติธรรมไทย 3 ประเภท: แพ่ง / อาญา / ปกครอง
