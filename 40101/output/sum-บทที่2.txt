★ ส่วนที่ 1: นิติปรัชญาคืออะไร (ออกสอบบ่อย)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔍 นิติปรัชญา

  - เป็นส่วนหนึ่งของปรัชญา ศึกษากฎหมายในเชิงนามธรรมชั้นสูง
  - ศึกษาเจาะลึกว่า "กฎหมายแท้จริงคืออะไร" ไม่ใช่เพียงกฎหมายส่วนใดส่วนหนึ่ง
  - ตอบคำถาม: กฎหมายมีผลบังคับได้อย่างไร? อะไรคือหลักในการบัญญัติกฎหมาย?

📚 เนื้อหาสำคัญ 3 ประการ
  - อภิปรัชญาทางกฎหมาย — กฎหมายแท้จริงคืออะไร?
  - ญาณวิทยาทางกฎหมาย — เรารู้กฎหมายได้อย่างไร?
  - จริยศาสตร์ทางกฎหมาย — กฎหมายควรเป็นอย่างไร?

❓ ทำไมต้องเรียนนิติปรัชญา?
  → เพื่อเข้าใจที่มา ความชอบธรรม และบทบาทของกฎหมายในสังคม
  → ช่วยให้ตีความและบังคับใช้กฎหมายได้อย่างถูกต้องและมีหลักการ

★ ส่วนที่ 2: สำนักความคิดทางกฎหมาย (ออกสอบมาก)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚡ สำนักที่ 1: กฎหมายธรรมชาติ (Natural Law)

  หลักคิด:
  - กฎหมายมีพื้นฐานบนศีลธรรมโดยธรรมชาติ ค้นพบได้ด้วยเหตุผล
  - กฎหมายที่ขัดกับสิทธิธรรมชาติถือว่าไม่ชอบด้วยกฎหมาย

  นักคิดสำคัญ:
  จอห์น ล็อก (John Locke)
  → "ชีวิต เสรีภาพ และทรัพย์สิน" คือสิทธิธรรมชาติหลัก
  → หากรัฐบาลละเมิดสิทธิ ประชาชนมีสิทธิโค่นล้มรัฐบาล

  ชาร์ล รุสโซ (J.-J. Rousseau)
  → มนุษย์ดีโดยธรรมชาติ แต่สังคมทำให้เกิดความไม่เท่าเทียม
  → "เจตนารมณ์ร่วม" (General Will) = เจตนาโดยรวมเพื่อประโยชน์ส่วนรวม
  → ประชาชนอาจ "ถูกบังคับให้เป็นอิสระ" หากฝ่าฝืนเจตนารมณ์ร่วม

⚡ สำนักที่ 2: นิติปฏิฐานนิยม (Legal Positivism)

  หลักคิด:
  - กฎหมายเป็นกฎเกณฑ์ที่มนุษย์สร้างขึ้น ไม่ต้องสอดคล้องกับศีลธรรม
  - กฎหมายถูกหรือผิดขึ้นอยู่กับกระบวนการตรา ไม่ใช่เนื้อหาทางศีลธรรม

  นักคิดสำคัญ:
  เบนแธม (Bentham)
  → กฎหมาย = คำสั่งของรัฏฐาธิปัตย์ที่มีบทลงโทษ
  → เป้าหมาย: ความสุขสูงสุดของคนจำนวนมากที่สุด (Utilitarianism)
  → วิจารณ์ Common Law ว่าเป็น "Dog Law" — ตัดสินย้อนหลัง

  จอห์น ออสติน (John Austin)
  → กฎหมาย = คำสั่งของรัฏฐาธิปัตย์ + บทลงโทษ
  → แยกกฎหมายออกจากศีลธรรมอย่างชัดเจน

⚡ สำนักที่ 3: ประวัติศาสตร์ (Historical School)
  - กฎหมายเกิดจากจิตวิญญาณของชาติ (Volksgeist) สะสมผ่านประวัติศาสตร์
  - นักคิดสำคัญ: ซาวิญีย์ (Savigny)

⚡ สำนักที่ 4: นิติศาสตร์เชิงสังคมวิทยา (Sociological Jurisprudence)
  - ศึกษากฎหมายจากความเป็นจริงในสังคม ไม่ใช่แค่ตัวบท
  - กฎหมายต้องปรับตามสภาพสังคมที่เปลี่ยนแปลง

⚡ แนวคิดนิติปรัชญาสมัยใหม่
  - ผสมผสานเพื่อแก้ปัญหาสังคมยุคใหม่ รวมสิทธิมนุษยชน ความเท่าเทียม

★ ส่วนที่ 3: นิติศาสตร์ 3 แขนง
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  นิติศาสตร์โดยแท้ (Positive Law)
  → ศึกษากฎหมายที่ใช้บังคับในปัจจุบัน (เนื้อหา + นิติวิธี)

  นิติศาสตร์ทางข้อเท็จจริง
  → ประวัติศาสตร์กฎหมาย สังคมวิทยากฎหมาย

  นิติศาสตร์ทางคุณค่า
  → กฎหมายเปรียบเทียบ, นิติบัญญัติ

  - นิติปรัชญา = ปรัชญาของกฎหมาย ศึกษาเชิงนามธรรมชั้นสูง
  - สำนักกฎหมายธรรมชาติ: กฎหมายต้องสอดคล้องกับศีลธรรม
  - นิติปฏิฐานนิยม: กฎหมาย = คำสั่งของรัฏฐาธิปัตย์
  - จอห์น ล็อก: ชีวิต เสรีภาพ ทรัพย์สิน = สิทธิธรรมชาติ
  - รุสโซ: เจตนารมณ์ร่วม (General Will)
